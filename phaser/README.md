# Project-X Phaser Pivot: Starlane Run

A lightweight Phaser prototype inspired by **Slay the Spire** + **FTL**:
- Branching run map (combat / elite / repair / shop / event / boss)
- Turn-based ship combat with a card hand, energy, block, weak/vulnerable
- Run progression: HP carries over, scrap economy, post-combat card rewards

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Build output is configured for `../docs` (GitHub Pages style).
